# أَوزَان — مُحَلِّل العَرُوض العربي

إنَّهُ تطبيق متعدد المنصات مبني على (فلاتر) لتحليل عَروض الشعر العربي العَمُودِي. أدخِل بيتًا من الشعر لِيُحدِّدَ التطبيق وزنه، ويَعرض تفاعيله، ويُسَمِّي بحره؛ أو اكتِب ما شِئتَ باللُّغة العربية وسيحاول التَطبِيق إيجاد تفاعيل تُلائِم ما كَتَبت، ويُخبِرُكَ -إن شاءَ الله- في حالَ وُجُودِ أخطاء تَمنَعهُ من الوزن.

---

## المميزات

- **تحليل عَروضي فوري** — يُحلِّل الكلام أثناء كِتابَتِه ويَكتب تَقرِيرَ التحليل.
- **وزن الكلام والأبيات** — يُعطي وَزنًا للكلام سواء أكانَ بيتَ شِعرٍ أم لا.
- **تمييز الأخطاء في النص** — يُشِيرُ بالألوانِ للمخالفات العَروضية (كالتقاء الساكنين) أثناء الكتابة.
- **قواعد الكلمات الخاصة** — يُجِيدُ قراءة الحروف التي لا تُكتَب كما في: الله، الذي، الرحمن، هذا، لكن.
- **تخطيط الصدر والعجز** — يُعرض كل بيت على هيئة صدر وعجز، ويُكَيِّفُهما مع حجم الشاشة.
- **واجهة مبنية للعربية** — الواجهة بالكامل من اليمين إلى اليسار مع خط شهرزاد العربي.
- **متعدد المنصات** — يعمل في الويب ويعمَلُ تطبيقًا في (أندرويد) و(آي أوه أس) و(ويندوز) و(ماك).

## البحور

يبدُو لِي أن التطبيق مُستَعِد لِيستَوعِب جميع البحور الشعرية المُستَعملةِ والمُهملَة، إلَّا أني لم أُضِف غيرَ ما أرَدتُ تَجرِبَتَه، ورَكَّزتُ على أُمُورٍ أخرى. ما يَلِي جدولُ البُحُورِ التي أضفتها وتفعِيلاتِها، سيزدادُ الجدولُ -إن شاء الله- متى تَسَنى لي الإضافة فيهِ أو ساهَمَ أحدُهُم بإضافة بحر.

| البحر | التفعيلة الأساسية |
| --- | --- |
| الطويل | فعولن مفاعيلن فعولن مفاعلن |
| البسيط | مستفعلن فاعلن مستفعلن فاعلن |
| الوافر | مفاعلتن مفاعلتن مفاعلتن |
| مخلع البسيط | مستفعلن فاعلن فعولن |

يمكن إضافة بحور جديدة بتوسيع ملف `lib/core/buhoor.dart`.

## آلية العمل

```
النص العربي
    │
    ▼
المحلل اللغوي /   (lib/core/parser/token_stream.dart)
    │  يحوّل الأحرف إلى رموز: حرف / حركة / مسافة
    ▼
الشجرة التفصيلية   (lib/core/parser/ast.dart)
    │  يبني شجرة عُقَد مترابطة من الأحرف والحركات والمسافات
    ▼
مُطَبِّق القواعِد      (lib/core/rules/)
    │  يُحلِّل النص بناءً على قواعِد قابِلَة للتعديل والإضافة
    │  
    │  
    ▼
كاتب التقرير والنتائج     (lib/core/utils/prosody_writing/)
    │  يُرمَّز كل حرف بـ 1 (متحرك) أو 0 (ساكن)
    │   يُعطي قائمة الأخطاء وأماكن وجودها إن وُجِدَت
    │   ويستَنبِط سلسلات التفاعِيل اللائِقة للنص
    │   ويَبحث عن البحر الشعري المناسب للتفعيلات
    ▼
عرض النتائج                      (lib/verse.dart/lib/main.dart)
     الكتابة العَروضية · التفاعيل · اسم البحر · رسم الأخطاء
```

## البدء باستخدام المشروع

### المتطلبات الأساسية

- [Flutter SDK](https://docs.flutter.dev/get-started/install) بإصدار `^3.10.3` أو أحدث
- Dart SDK بإصدار `^3.10.3` أو أحدث

### تثبيت المكتبات

```bash
flutter pub get
```

### التشغيل

```bash
# الأجهزة المحمولة
flutter run

# سطح المكتب
flutter run -d linux
flutter run -d macos
flutter run -d windows
```

### البناء

```bash

flutter build apk

# iOS
flutter build ios

# Linux
flutter build linux

# macOS
flutter build macos

flutter build windows
```
